# Repeat string in JavaScript:
Repeat string in JavaScript:
* https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/repeat
```js
"abc".repeat(-1); // RangeError
"abc".repeat(0);  // ''
"abc".repeat(1);  // 'abc'
"abc".repeat(2);  // 'abcabc'
```
